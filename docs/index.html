『Mad Master』は…

ゲーム制作におけるマスターデータ管理を支援するツールです

Microsoft Store で入手

特徴

  • スプレットシートを使用した煩わしい初期設定の排除を目的としています
  • 指定したデータ型（文字列、数値、日付、時刻、参照）の不一致、空白、重複などのバリデーション機能を搭載しています
  • サブテーブルを利用することで、スパースなテーブルを回避します
  • カラム、レコード、テーブルの並び替えをドラッグによって直感的に編集できます
  • エクスポートは、JSON、CSV形式に対応しています
